Interfaces
  InterfaceDescription
ICommand
Target ID and Move pair that represents a single command.
IController
ICommand generator via user inputs or reactions.
ICoordinates
A X-Y pair representing a point in the 2D space.
IGameBoard
Interface for work with game-board
IGamePieceFactory
Abstract factory holding the sole responsibility of creating instances of the various types of IPiece objects.
IGamePieceObserver
Interface for game piece movement notification

Implements observer design pattern.

IPiece
Game object controlled by the players and moved on the IGameBoard via IController.
IRenderable
Represents an element that is able to be processed by the video renderer.
IRenderer
Represents a renderer able to process IRenderable objects.
